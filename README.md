offpr connects different webhooks together, e.g.: Github and Gitlab

## Usage

    offpr -D ROUTE='/webhook' GITHUB_ACCESS_TOKEN='zxcv' GITLAB_ACCESS_TOKEN='asdf' GITHUB_ACCESS_TOKEN='qwerty' 

You can repeat this multiple times.
